# Support

Use GitHub Issues for reproducible bugs and feature requests. Include the application
version, Windows 11 build, local/remote mode, operation name, update source, exact
UpdateID/revision when relevant, and a redacted log excerpt.

Do not include credentials, private hostnames, SMTP settings, or unredacted update
history. Windows Update Agent or operating-system defects may need reproduction with
Microsoft-supported tools before they can be isolated from this application.

This project provides no warranty or guaranteed response time.
