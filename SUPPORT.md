# Support

Use GitHub Issues for reproducible GUI bugs and feature requests. Include the GUI
version, Windows 11 build, local/remote mode, selected cmdlet and parameter set, and a
redacted log excerpt.

Questions or defects in PSWindowsUpdate command behavior should be reproduced in
Windows PowerShell and reported to the
[upstream project](https://github.com/mgajda83/PSWindowsUpdate) when appropriate.

This project provides no warranty or guaranteed response time.
